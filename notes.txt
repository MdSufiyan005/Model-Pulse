

## 🛡️ nginx Reverse Proxy (Optional)

```nginx
server {
    listen 8000;
    client_max_body_size 0;

    location / {
        proxy_pass         http://127.0.0.1:8001;
        proxy_read_timeout 300s;
        proxy_send_timeout 300s;
    }
}
```

Run uvicorn on port 8001, nginx on 8000.

> **llama-cpp-python on ARM:** Follow the [official build guide](https://github.com/abetlen/llama-cpp-python) for RPi / Jetson. Pre-built wheels exist for common targets.
